# BookReader
#### 任阅小说阅读器。实现追书推荐、标签检索、3D仿真翻页效果、文章阅读、缓存章节、日夜间模式、文本朗读等功能。

## 项目
本项目基于RxJava + Retrofit2 + Dagger，项目中API均来自追书神器，纯属共享学习之用，不得用于商业用途。
